## 1. Install

```bash
yarn install
```

## 2. Run Server

```bash
yarn server:start
```

## 3. Run app

```bash
yarn preview
```

go to <http://localhost:4173>

## Test

```bash
yarn test
```

## Coverage report

```bash
yarn coverage
```
